\chapter{基本框架}

我们首先介绍\textbf{相对论性量子场论}的核心思想和工具，主要以\textbf{标量场}为例。QFT 这一简称其实多数也是指相对论性的，实际上继承了经典相对论的时空连续统。时空的连续性有可能是未来最完备的量子引力理论必须严肃讨论的问题，在此处则受到默认。

本节开始，默认读者具有较好的相对论基础。当然，我们也将简单回顾，但细节和哲思均可从笔者的《引力》第 1.1 节查到。
时空坐标为 $x^\mu$，这里 $\mu=0,1,2,3$，$x^0=ct$ 表示坐标时间。
此后均取单位制 $\hbar = c = 1$，置闵氏时空惯性系。
狭义相对论相当于取闵氏度规 $\d s^2=\eta_{\mu\nu}\d x^\mu\d x^\nu$ 且 $\{x\}$ 为惯性系。内积为 $v_\mu w^\mu = \eta_{\mu\nu}v^\mu w^\nu$。注意使用了求和约定。
但不同于广义相对论(GR)的习惯，QFT通常约定
\[\eta_{\mu\nu}=\diag(1,-1,-1,-1),\]
号差为 $\eta^\mu{}_\mu = 1 - 1 \times 3 = -2$。
相空间上可以建立拉氏、哈氏理论。位形坐标 $q$ 对应的共轭动量为 $p=\pdv*{L}{\dot q}$。如果取惯性坐标 $x^\mu$，则（自由作用量下）共轭动量就是普通的 4 维线动量 $p^\mu$。
在时空上可以建立场的拉氏理论。它相当于将分析力学的广义坐标 $q$ 替换为场量 $\Phi$，路径 $q(t)$ 变为时空场 $\Phi(x)$。故（不含三阶导及以上的）作用量、E-L 方程为
\eq{
    S[\Phi]=\int\d[4]{x}\Lag(\Phi,\p_\mu\Phi)\im
    \pdv{\Lag}{\Phi}-\p_\mu\pdv{\Lag}{\p_\mu\Phi}=0.
}
$\Lag$ 叫做拉氏密度，与拉氏量的关系为 $L=\int\d[3]{x}\Lag$。同理，哈氏密度、共轭动量密度为
\eq{
    \Ham=\Pi(x)\dot\Phi(x) - \Lag,\quad
    \Pi(x)=\pdv{\Lag}{\dot\Phi(x)}.
}
这里点表示 $\p_t$。
若真准确地说，可以先设置离散点系，随后用连续极限来过渡到场。因此我们常说场是无穷自由度的。换言之，把有限自由度系统推广到无穷多时空点的自由度，就会得到场论。所以某种意义上（量子）多体理论也是（量子）场论。
这里不要用离散点系中的有限自由度理解为粒子或质点，它们单纯是一些时空点。无穷自由度的意思单纯是因为时空点集 $\R^4$ 是不可数集。
这些场的分量在经典力学中无非是实数。QFT就是将场提升\textbf{量子场}，表为\textbf{场算符}，此后算符默认省略 $\hat{}$ 帽号，读者可自明。场算符的本征值、期望值可以联系回经典理论。“波函数”现在提升为某种本征场的波函数。所以注意不要把场和波函数弄混，场对应的是广义坐标。

场算符有基本的动机。我们已经看到，上一节的非相对论量子力学中，粒子数是固定的，但实验告诉我们粒子一定存在湮灭和产生现象。我们只得将希望寄托于相对论性的量子力学\footnote{注意，所谓的“非相对论性”是因为我们从经典力学做量子化而来。尝试从相对论、经典场论做对易关系便也就考虑了“相对论性”量子力学，仍然满足上一节的那些公理。}。相对论和经典场论的关系十分紧密，换言之，时空场是必定会引入的，故进一步再考虑场的算符化即可。
后续将了解到，粒子可以解释为\textbf{场算符的激发态}，首先不同激发态可测得不同粒子数，叠加时测量数值也具有概率性。总之，粒子数将可变。这种方法不仅简化了多粒子系统的描述，还为理解粒子之间的相互作用提供了一个更自然的框架。

直接将时空场算符化所得到的公式，可以直接继承相对论显式带来的 Lorentz 协变性。场算符的时变性意味着 QFT 几乎只使用 Heisenberg 和 Dirac 绘景。使用 Schr\"odinger 绘景的话，需要讨论态演化的 Schr\"odinger 方程，比较繁琐。转化到 Heisenberg 绘景上来，其实直接就是经典场论的那些场运动方程的量子版本。

路径积分形式对QFT尤其重要。无非是替换 $q(t)\to\Phi(x)$：
\[\int\mathcal D q\,\e^{\frac{\i}{\hbar}S[q]} \to \int\mathcal D\Phi\,\e^{\frac{\i}{\hbar}S[\Phi]}.\]
前者对所有粒子轨道求和，后者对所有场的历史求和。这显式上避开了算符或态矢。

\section{正则量子化}
量子算符化的关键在于给出对易关系。
将 $\Phi(x),\Pi(x)$ 量子化，就是要先把 Heisenberg 绘景下离散自由度 $q_r(t),p_r(t)$ 的对易关系推广为无穷多自由度。
首先，不同离散自由度不应该相互影响，正则算符需要满足的等时对易关系为
\eq{
    \tcbhighmath{
        [q_r(t),p_s(t)]=\i\hbar\delta_{rs},
    \quad
    [q_r(t),q_s(t)]=[p_r(t),p_s(t)]=0.
    }
}
《引力》1.1 节提过，可先将整个空间离散化，划分成无穷多个体积元 $V_i$，再取 $V_i\to0$ 的极限来得到连续空间的结果。在体积元 $V_i$ 中，定义相应的广义坐标
\begin{equation}
  \Phi_i(t)\equiv \frac{1}{V_i}\int_{V_i}\d^3x\,\Phi(\bm{x},t),
\end{equation}
这是场 $\Phi(\bm{x},t)$ 在 $V_i$ 中的平均值。将 $\partial_\mu\Phi$ 和拉格朗日量密度 $\mathcal{L}(\Phi,\partial_\mu\Phi)$ 在 $V_i$ 中的平均值记为
\begin{equation}
  \partial_\mu\Phi_i \equiv \frac{1}{V_i}\int_{V_i}\d^3x\,\partial_\mu\Phi,
  \qquad
  \mathcal{L}_i \equiv \frac{1}{V_i}\int_{V_i}\d^3x\,\mathcal{L}(\Phi,\partial_\mu\Phi),
\end{equation}
当 $V_i\to0$ 时，$\mathcal{L}_i$ 成为 $\Phi_i$ 和 $\partial_\mu\Phi_i$ 的函数 $\mathcal{L}_i(\Phi_i,\partial_\mu\Phi_i)$。拉格朗日量表达为
\[
  L(t)
   = \int\d^3x\,\mathcal{L}
   = \sum_i\int_{V_i}\d^3x\,\mathcal{L}
   = \sum_i V_i\frac{1}{V_i}\int_{V_i}\d^3x\,\mathcal{L}
   = \sum_i V_i\mathcal{L}_i(\Phi_i,\partial_\mu\Phi_i).
\]
体积元 $V_i$ 中的广义动量是\footnote{此处未使用 Einstein 求和约定。}
\begin{equation}
  P_i(t)
  = \pdv{L}{(\partial_0\Phi_i)}
  = \sum_j V_j\pdv{\mathcal{L}_j}{(\partial_0\Phi_i)}
  = \sum_j V_j\delta_{ji}\pdv{\mathcal{L}_i}{(\partial_0\Phi_i)}
  = V_i\pdv{\mathcal{L}_i}{(\partial_0\Phi_i)}.
\end{equation}
根据 (2.45) 式，等时对易关系为
\begin{equation}
  [\Phi_i(t),P_j(t)] = \i\delta_{ij},
  \qquad
  [\Phi_i(t),\Phi_j(t)] = 0,
  \qquad
  [P_i(t),P_j(t)] = 0.
\end{equation}
引入
\begin{equation}
  \Pi_i(t)\equiv \pdv{\mathcal{L}_i}{(\partial_0\Phi_i)}
  = \frac{P_i(t)}{V_i},
\end{equation}
则第一条和第三条等时对易关系可用 $\Pi_i(t)$ 表达为
\begin{equation}
  [\Phi_i(t),\Pi_j(t)] = \i\frac{\delta_{ij}}{V_j},
  \qquad
  [\Pi_i(t),\Pi_j(t)] = 0.
\end{equation}
离散指标连续化时，Kronecker 符号将变成 Dirac 函数。
设 $f_i$ 是 $f(\bm{x})$ 在 $V_i$ 上的平均值，有
\begin{equation}
  f_i = \sum_j f_j\delta_{ij}
      = \sum_j V_j f_j\frac{\delta_{ij}}{V_j}.
\end{equation}
在 $V_i\to0$ 时的极限形式为
\begin{equation}
  f(\bm{x})=\int\d^3y\,f(\bm{y})\delta(\bm{x}-\bm{y}),
\end{equation}
也就是说，在连续极限下，
\begin{equation}
  \frac{\delta_{ij}}{V_j}\to \delta(\bm{x}-\bm{y}).
\end{equation}
另一方面，在此极限下，$\Phi_i(t)\to\Phi(\bm{x},t)$，$\partial_\mu\Phi_i\to\partial_\mu\Phi(\bm{x},t)$，$\mathcal{L}_i\to\mathcal{L}(\bm{x},t)$，从而 $\Pi_i(t)$ 变成由 (1.168) 式定义的共轭动量密度，
\begin{equation}
  \Pi_i(t)=\pdv{\mathcal{L}_i}{(\partial_0\Phi_i)}
  \to \pdv{\mathcal{L}}{(\partial_0\Phi)}
  = \Pi(\bm{x},t).
\end{equation}
因此，等时对易关系 (2.50) 和 (2.52) 化为
\begin{equation}
    \tcbhighmath{
  [\Phi(\bm{x},t),\Pi(\bm{y},t)] = \i\delta(\bm{x}-\bm{y}),
  \quad
  [\Phi(\bm{x},t),\Phi(\bm{y},t)] = 
  [\Pi(\bm{x},t),\Pi(\bm{y},t)] = 0.
    }
\end{equation}
推广到包含若干个场 $\Phi_a$ 的系统，假设不同的场相互独立，则
\begin{equation}
  [\Phi_a(\bm{x},t),\Pi_b(\bm{y},t)] = \i\delta_{ab}\delta(\bm{x}-\bm{y}),
  \quad
  [\Phi_a(\bm{x},t),\Phi_b(\bm{y},t)] = 
  [\Pi_a(\bm{x},t),\Pi_b(\bm{y},t)] = 0.
\end{equation}
这就是量子场论中的正则对易关系，它是场的正则量子化的出发点。此时，系统的正则变量 $\Phi_a(\bm{x},t)$ 和 $\Pi_a(\bm{x},t)$ 都是 Hilbert 空间上的算符。

\section{自由标量场}
\subsection{平面波展开}
考虑最简单的实标量场 $\phi$（的量子化），由于它是实数（算符），故满足自伴性：
\[ \phi =  \phi^\dagger.\]
场是粒子系统连续化，动能项显然用 $\p_\mu\phi$ 构造。
最简单的选择是 $\p^\mu\phi\p_\mu\phi$。
为了保证 $\dot\phi^2$ 系数为正，-2 号差下 $\p^\mu\phi\p_\mu\phi$ 系数应为正，可取
\[\Lag= \frac 12\p^\mu\phi\p_\mu\phi-U(\phi)\im \square\phi = -\dv{U}{\phi},\]
这里 $\square=\p^\mu\p_\mu = \p_t^2 - \nabla^2$ 为 d'Alembert 算子，$\nabla^2$ 为 Laplace 算子。
$U(\phi)$ 可展为 $\phi$ 的多项式。一次项可通过平移收进新定义的 $\phi$ 中，从 E-L 方程也可看出 $\phi$ 一次项无实质贡献。
若规定\textbf{$Z_2$ 变换} $\phi'=-\phi$ 下不变，这属于内部对称性，则 $U(\phi)$ 是偶函数，奇数次项也不会出现，故有时也写成 $U(\phi^2)$。这属于分立对称性。
最简单取法是\textbf{谐振子模型} $U(\phi)=\frac 12 m^2\phi^2$，得\textbf{Klein-Gordon 方程}
\eq{
    (\square+m^2)\phi=0.
}
从而哈氏密度 $\Ham = \frac 12 (\dot\phi^2 + (\grad\phi)^2 + m^2\phi^2) \geqslant 0$ 有下界。
这是线性方程。线性方程具有平面波解 $\e^{-\i px}$。其中简写 $px:=p_\mu x^\mu=Et-\bm p\vdot\bm x$，$p^\mu$ 是常数 4-动量，不是算符。
对于 $\phi=\e^{-\i px}$，$\i\p^\mu\phi = p^\mu\phi$，$\square\phi = -p_\mu p^\mu\phi$，代入 $(\square+m^2)\phi=0$ 得相对论性色散关系 $p^2:=p_\mu p^\mu=m^2$。可见 $p^\mu$ 确为 4-动量，在 $\hbar=1$ 单位制下不用区分动量和波矢 $p^\mu=k^\mu$。
$p^\mu$ 是\textbf{4-动量算符} $\hat p^\mu=\i\p^\mu$ 的本征值，并且
$\square=-\hat p^2$。$\hat p^0 = \i\pdv{t} = \hat E,\hat{\bm p} = -\i\grad$ 分别是能量、动量算符。这就是选择 $-2$ 号差的原因。平面波解往往对应着\textbf{自由}情形。作为对比，我们知道自由 Schr\"odinger 方程具有平面波解，而得非相对论性色散关系 $T=\bm p^2/2m$。
即使将不同动量的平面波叠加，每个平面波仍然独立演化，之间无相互作用，故称自由。

实际上，一般解显然就是平面波的 Fourier 展开，对动量模式连续地积分。
这里可对任意 $1\sim 4$ 个变量做 Fourier 变换，只是三维或四维最方便。
三维方法类似于波函数的方法。先取 $\phi(x) = \int\frac{\d[3]p}{(2\pi)^3}\e^{\i \bm p\cdot\bm x}\phi_{\bm p}(t)$，代回方程：
\[(\square+m^2)\phi = \int\frac{\d[3]p}{(2\pi)^3}(\dot\phi_{\bm p}(t) + \phi_{\bm p}(t) (\bm p^2 + m^2) )\e^{\i\bm p\cdot\bm x} = 0 \iff \dot\phi_{\bm p}(t) + \phi_{\bm p}(t) (\bm p^2 + m^2) = 0,\]
故对每个模式 $\bm p$ 满足同样形式的方程，所以谐振子模型的确是自由模型。
若引入 $\frac{\lambda}{4!}\phi^4$ 等，则方程 $(\square+m^2)\phi = \frac{\lambda}{3!}\phi^3$ 不再线性，则动量模式之间会相互作用，不再“自由”。
首先，单独的平面波 $\e^{-\i px}$ 不是解，$p$ 通过 $\phi^3$ 项激发新模式 $3p$，存在自相互作用。
两平面波叠加还将通过 $\phi^3$ 项产生 $2p_1+p_2,p_1+2p_2$，因此模式 $p_1$ 的存在影响了 $p_2$，存在相互作用。
标量场的相互作用会在之后再讨论，本节只讲自由标量场。


对固定的 $\bm p$ 下，这是二阶常微分方程，解空间是二维的。注意特解 $\e^{\pm\i E_{\bm p} t}$，其中
\eq{
    E_{\bm p}:=\sqrt{\bm p^2+m^2}>0.
}
$\e^{-\i E_{\bm p} t},\e^{\i E_{\bm p} t}$ 是两个线性独立的特解，故可作为解空间基底，通解形如 $A_{\bm p} \e^{-\i E_{\bm p} t} + B_{\bm p} \e^{\i E_{\bm p} t}$，代回有
\eq{
    \phi(x) = \int\frac{\d[3]p}{(2\pi)^3} (A_{\bm p} \phi^+_{\bm p}(x) + B_{\bm p} \phi^-_{\bm p}(x) ),\quad \phi^\pm_{\bm p} = \e^{\mp\i E_{\bm p} t}\e^{\i \bm p\cdot\bm x} = \e^{\mp\i E_{\bm p}t + \i\bm p\cdot\bm x}.
}
平面波 $\e^{-\i px}=\e^{-\i E_{\bm p}t + \i\bm p\cdot\bm x}$ 具有正能，因为这里 $p^0=E_{\bm p}>0$。而$\e^{-\i qx}=\e^{\i E_{\bm p}t + \i\bm p\cdot\bm x}$ 具有负能，因为这里 $q^0=-E_{\bm p}<0$。
下面考虑都转化为 $p^0>0$ 正能形式。
注意 $E_{\bm p}=E_{-\bm p}$，因此 $\phi^\pm_{\pm\bm p}=\e^{\mp\i px}$。并且对被积函数做变换 $\bm p\to-\bm p$，三维动量空间积分 $\int \d[3]{p} B_{\bm p}\phi^-_{\bm p}$ 不变，即 $\int \d[3]{p} B_{\bm p}\phi^-_{\bm p} = \int \d[3]{p} B_{-\bm p}\e^{\i px}$（绝对值测度）。综上，通解一定形如
\[
    \phi(x) = \int\frac{\d[3]p}{(2\pi)^3}\frac{1}{\sqrt{2E_{\bm p}}} (a_{\bm p} \e^{-\i px} + \tilde a_{\bm p} \e^{\i px} ),\quad px = \sqrt{\bm p^2+m^2}\, t - \bm p\cdot \bm x.
\]
这里分母提出系数 $\sqrt{2E_{\bm p}}$ 的原因稍后介绍。实标量场满足 $\phi^\dagger=\phi$，对比系数知 $\tilde a^\dagger_{\bm p}=a_{\bm p}$，因此两个系数 $a_{\bm p},\tilde a_{\bm p}$ 不再是独立无关的。最终
\eq{
    \tcbhighmath{\phi=\int\frac{\d[3]p}{(2\pi)^3}\frac{1}{\sqrt{2E_{\bm p}}}\left(a_{\bm p}\e^{-\i px}+a^\dagger_{\bm p}\e^{\i px}\right).}
}
以上代数计算实际上对经典的场也成立，并没有要求场是算符的。当然，算符的共轭转置要对应为数字的复共轭。

共轭动量密度算符的平面波展开式为
\begin{equation}
  \Pi(\bm{x},t)=\pdv{\Lag}{\partial_0\phi}=\partial_0\phi(\bm{x},t)
  = \int\frac{\d^3p}{(2\pi)^3}\frac{-\i p^0}{\sqrt{2E_{\bm{p}}}}
    \left(a_{\bm{p}}\e^{-\i p x}
    -a^\dagger_{\bm{p}}\e^{\i p x}\right).
\end{equation}
足以见得 $\phi,\Pi$ 均可用 $a_{\bm p},a_{\bm q}^\dagger$ 表示，如果全部换成 $a_{\bm p},a_{\bm q}^\dagger$ 语言，我们就要用 $\phi,\Pi$ 的对易关系来计算 $a_{\bm p},a_{\bm q}^\dagger$ 的对易关系。提取系数显然需要 Fourier 变换。
结果是
\eq{
    \tcbhighmath{
    [a_{\bm p},a_{\bm q}^\dagger]
    =
    (2\pi)^3\delta(\bm p-\bm q),
    \quad
    [a_{\bm p},a_{\bm q}]=[a_{\bm p}^\dagger,a_{\bm q}^\dagger]=0.
    }
}
故之所以提出系数 $\sqrt{2E_{\bm p}}$，是为了让对易关系好看。
有时甚至为了对易关系更简单可以将 $(2\pi)^3$ 也收进展开系数的定义中。
但稍后将证明积分元 ${\d[3]p}/{E_{\bm p}}$ 才 Lorentz 不变，因此系数 $a_{\bm{p}}$ 并非 Lorentz 不变。
为了 Lorentz 不变可以进行重标度：
\[\phi=\int\frac{\d[3]p}{(2\pi)^3 2E_{\bm p}}\left(\alpha(\bm p)\e^{-\i px}+\alpha^\dagger(\bm p)\e^{\i px}\right).\]
表述等价，只不过对易关系变成稍复杂的 $[\alpha(\bm p),\alpha^\dagger(\bm q)]=(2\pi)^3 2E_{\bm p} \delta(\bm p-\bm q)$。

下面证明 ${\d[3]p}/{E_{\bm p}}$ Lorentz 不变。
考虑完整的四维动量空间，由于 4-动量是 Lorentz 矢量，变换形如 4-坐标 Lorentz 变换，因此 $\d[4]{p}$ 和 $\d[4]x$ 一样 Lorentz 不变。
所谓的三维积分，其实就是四维积分在超曲面上的诱导。由前文知动量模式要满足在壳条件 $E_{\bm p}^2=\bm p^2+m^2$，换言之，三维积分在动量空间的超曲面
\eq{
    p^2=m^2.
}
不过这个双曲面覆盖两个分支，只考虑正能形式其实就是想办法转化到 $p^0>0$ 分支。
$\d[4]{p}$ 诱导出双曲面单支上的 Lorentz 不变体元，方法是利用 Dirac 函数和阶跃函数来挑选超曲面
\[
    \int\d[4]{p}\delta(p^2-m^2)\theta(p^0)= \int_{p^2=m^2}\d[3]{p} \sum_{p^0=\pm E_{\bm p}}\frac{\theta(p^0)}{2| p^0|}=\int_{p^2=m^2,p^0>0}\frac{\d[3]p}{2E_{\bm p}}.
\]
注意这里用到 $\delta(x)$ 的复合函数性质：考虑连续实函数 $g(x)$，若方程 $g(x) = 0$ 具有若干个分立的单根 $x_i$，则
\begin{align*}
    \int\d{x}f(x)\delta(g(x))&=\sum_i\int_{O_i\ni x_i}\d{x}f(x)\delta(g(x))=\sum_i\int_{O_i\ni x_i}\frac{\d{g}}{|g'(x)|}f(x)\delta(g)\\
    &=\sum_i\frac{f(x_i)}{|g'(x_i)|} = \int\d{x} f(x)\sum_i\frac{\delta(x-x_i)}{|g'(x_i)|},\\
    \iff& \tcbhighmath{\delta(g(x))=\sum_i \frac{\delta(x-x_i)}{|g'(x_i)|}.}
\end{align*}
由于 $p^2-m^2$ 为标量，$\delta(p^2-m^2)$ 直接不变；由正时性、矢量类时性知 $\sgn(\Lambda^0{}_\mu p^\mu)=\sgn(p^0)$，从而 $\theta(p^0)=\theta(p'^0)$ 也不变。故 ${\d[3]p}/{E_{\bm p}}$ 不变。
注意三维积分区域是双曲面单支，但坐标化其实即 $\R^3$，故通常省略 $p^2=m^2,p^0>0$ 记号。
实际上，从四维展开方法来看待也更清晰：将 $\phi=\int\frac{\d[4]{p}}{(2\pi)^4}\e^{\i px}\tilde\phi(p)$ 代入方程有
\[(\square + m^2)\phi = \int\frac{\d[4]{p}}{(2\pi)^4}\tilde\phi(p)(-p^2 + m^2)\e^{\i px}=0\iff (-p^2+m^2)\tilde\phi(p)=0,\]
因此 $\tilde{\phi}(p)$ 至多在壳时非零，但积分又是非零的有限值，则它必须正比于一维 $\delta$ 函数：$\tilde{\phi}(p) = 2\pi \delta(p^2 - m^2) f(p)$，其中 $f(p)$ 是复函数，$2\pi$ 只是为了最终约掉分母的一个 $2\pi$。将其代回 $\phi(x)$，注意 $-p^2+m^2=-(p^0)^2 + E_{\bm p}^2$，
有
\begin{align*}
    \phi(x)&=\int\frac{\d[4]{p}}{(2\pi)^3}\e^{\i px} \delta(p^2 - m^2) f(p)= \int_{p^2=m^2}\frac{\d[3]p}{(2\pi)^3}\sum_{p^0=\pm E_{\bm p}}\frac{\e^{\i px}f(p)}{2|p^0|} \\
    &=\int_{p^2=m^2}\frac{\d[3]p}{(2\pi)^32E_{\bm p}}\left(f(E_{\bm p},\bm p)\phi^+_{\bm p}(x)+f(-E_{\bm p},\bm p)\phi^-_{\bm p}(x)\right),\\
    &=\int_{p^2=m^2,p^0>0}\frac{\d[3]p}{(2\pi)^3 2E_{\bm p}}\left(f(p)\e^{-\i px}+f(-p)\e^{\i px}\right).
\end{align*}
最后一个等号是因为对第二项取 $\bm p\to-\bm p$：首先同理 $\int \frac{\d[3]p}{E_{\bm p}}f(\pm E_{\bm p},\bm p)\phi^\pm_{\bm p}$ 在超曲面 $p^2=m^2$ 上关于 $\bm p\to-\bm p$ 不变，尝试把 $p^\mu$ 的积分超曲面锁定到 $p^0>0$，这样 $f(E_{\bm p},\bm p)=f(p)$，$f(-E_{\bm p},-\bm p)=f(-p)$。
四元函数 $f(p),f(-p)$ 独立\footnote{注意并没有给定函数 $f$，知道 $f(p)$ 当然不会确定 $f(-p)$，除非有对称性，比如奇函数性。}，分别正比于前文 $a_{\bm p},\tilde a_{\bm p}$。
这样直接可通过 $f(\pm E_{\bm p},\pm\bm p)\e^{\mp\i px},\phi$ 的不变性断言 ${\d[3]p}/{E_{\bm p}}$ 不变。






\subsection{粒子的湮灭与产生}

经过一些稍微长的计算得到自由场的哈氏量为
\begin{align}
  H
  &= \frac{1}{2}\int\frac{\d^3p}{(2\pi)^3}E_{\bm{p}}
     (a_{\bm{p}}a^\dagger_{\bm{p}}+a^\dagger_{\bm{p}}a_{\bm{p}}) = \frac{1}{2}\int\frac{\d^3p}{(2\pi)^3}E_{\bm{p}}
     (2a^\dagger_{\bm{p}}a_{\bm{p}}+(2\pi)^3\delta(\bm{p}-\bm{p}))\notag\\
  &= \int\frac{\d^3p}{(2\pi)^3}E_{\bm{p}}
     \left(N_{\bm{p}}+\frac{1}{2}(2\pi)^3\delta(\bm 0)\right),\quad N_{\bm{p}}:= a^\dagger_{\bm{p}}a_{\bm{p}},
\end{align}
其中


其中 $E_{\bm p}$ 由场的运动方程决定。
$a_{\bm p}^\dagger$ 产生一个动量为 $\bm p$ 的粒子，$a_{\bm p}$ 则湮灭一个动量为 $\bm p$ 的粒子。
于是，“粒子”不再作为基本点状对象预先放入理论，而是作为场的某个模式的激发出现。
同时，零点能项 $\frac{1}{2}\sum_r E_r$ 会变成发散的积分。这不是本节要解决的问题；在自由场量子化中通常先通过正规序或重整化处理这类真空能常数。

$N$ 称为\textbf{数算符}(number operator)。原因如下。
只要求出 $N$ 的本征值，就求出了能谱。
先看 $N$ 的基本性质。由 $[a,a^\dagger]=1$ 和对易子公式
\begin{align*}
  [AB,C] &= ABC-ACB+ACB-CAB = A[B,C]+[A,C]B,\\
  [A,BC] &= ABC-BAC+BAC-BCA = [A,B]C+B[A,C],
\end{align*}
可得
\begin{align}
    [N,a]
    &=
    [a^\dagger a,a]
    =
    a^\dagger[a,a]+[a^\dagger,a]a
    =
    -a,\\
    [N,a^\dagger]
    &=
    [a^\dagger a,a^\dagger]
    =
    a^\dagger[a,a^\dagger]
    +
    [a^\dagger,a^\dagger]a
    =
    a^\dagger.
\end{align}
若 $\ket n$ 是 $N$ 的归一化本征态，则
\[
    N(a\ket n)
    =
    (aN-a)\ket n
    =
    (n-1)a\ket n,\quad N(a^\dagger\ket n)
    =
    (a^\dagger N+a^\dagger)\ket n
    =
    (n+1)a^\dagger\ket n.
\]
所以 $a$ 会把数算符本征值降低 $1$，$a^\dagger$ 会把本征值升高 $1$，即 $a\ket{n}\propto\ket{n-1},a^\dagger\ket{n}\propto\ket{n+1}$。故又分别称为\textbf{湮灭算符}和\textbf{产生算符}。
通过以下方法可以确定系数。注意 $n + 1 = \mel{n}{aa^\dagger}{n},n=\mel{n}{a^\dagger a}{n}$，则系数确定为
\eq{
    a\ket{n} = \sqrt{n}\ket{n-1},\quad
    a^\dagger\ket{n} = \sqrt{n+1}\ket{n+1}.
}
然而注意本征值不能无限向下，因为对任意归一化态 $\ket\psi$，
\eq{
    \ev{N}{\psi}
    =
    \mel{\psi}{a^\dagger a}{\psi}
    =
    \ip{a\psi}{a\psi}
    \geq 0,
}
因此 $N$ 的本征值非负。若从某个本征态不断作用 $a$，本征值每次减 $1$，由于 $n=\ev{n}{n}=\ev{N}{n}\geq 0$。
必定存在一个最小本征值 $n_0$，它的本征态 $\ket{n_0}$ 满足 $a\ket{n_0}=0$，否则 $a\ket{n_0}$ 会给出本征值 $n_0-1$ 的态。
这是使本征值停止减小的条件。于是 $n_0=\ev{N}{n_0}=\ev{a^\dagger a}{n_0}=0$。换言之，最终必须停在最低态 $\ket0$。
这个最低态称为\textbf{基态}或\textbf{真空态}(vacuum)。
反之从 $\ket0$ 出发，所有其他态由反复作用 $a^\dagger$ 得到，所以 $N$ 的谱为非负整数，是离散谱的！离散谱的本征态一般确切地在 Hilbert 空间内，可以做归一化，而非广义本征态。
利用归一性，显然作用 $a^\dagger$ 可得所有激发态为
\eq{
    \ket n
    =
    \frac{1}{\sqrt{n!}}
    (a^\dagger)^n\ket0,
    \quad n=0,1,2,\cdots.
}
能量本征值为
\eq{
    H\ket n
    =
    E_n\ket n,
    \quad
    E_n=\hbar\omega\left(n+\frac{1}{2}\right).
}
这就是简谐振子的离散能谱。
最低能量非零，而是
\eq{
    E_0=\frac{1}{2}\hbar\omega.
}
这称为\textbf{零点能}(zero-point energy)或\textbf{真空能}。此非人为，而是由 $x,p$ 非对易性（或者说 $\hbar\ne 0$）导致。另一方面，也可注意湮灭产生算符与哈氏量的对易关系为
\eq{
    [a,H]=\hbar\omega a,
    \quad
    [a^\dagger,H]=-\hbar\omega a^\dagger.
}
作用于 $\ket n$ 有
\eq{
    H a\ket n=(aH-\hbar\omega a)\ket n=(E_n-\hbar\omega)a\ket n,\quad Ha^\dagger\ket n=(E_n+\hbar\omega)a^\dagger\ket n.
}
这种写法直接就说明能量是一份一份增加的。相邻能级差为 $E_{n+1}-E_n=\hbar\omega$。

可见，谐振子的这种升降算符的语言，立即变成清晰的物理图像：一个场模式的激发数就是对应动量粒子的粒子数。如果说\textbf{一次量子化}是指算符化并引入对易关系，引入产生、湮灭算符来描述粒子数可变的 Fock 空间就相对而言称为\textbf{二次量子化}。“正则量子化”是强调正则对易关系本身，故既可以指一次量子化，也可以指二次量子化。
注意，场论中的真空也不是“什么都没有”的状态，而是每个场模式都保留类似的零点涨落。

对比前文谐振子的 Heisenberg 表述可以预料，$a_{\bm{p}}$ 正是湮灭算符，；$a^\dagger_{\bm{p}}$ 正是产生算符，

简谐振子是最重要的模型。不只是因为可以精确求解，而是任意自由场做 Fourier 展开后，每个动量模式都像一个独立简谐振子。因此，理解简谐振子的正则量子化，就是提前理解自由量子场中一个模式如何变成一个粒子数自由度。随后，我们将发现湮灭和产生粒子的方法从中自然迸发。













类似地，对每个模式分别定义升降算符 $a_r,a_r^\dagger$，并可得
\eq{
    [a_r,a_s^\dagger]=\delta_{rs},
    \quad
    [a_r,a_s]=[a_r^\dagger,a_s^\dagger]=0.
}
为简洁起见把每个振子的质量取为 $1$，哈氏量为
\eq{
    H=\sum_r\left(
        \frac{p_r^2}{2}
        +
        \frac{1}{2}\omega_r^2q_r^2
    \right)
    =
    \sum_r
    \hbar\omega_r
    \left(
        a_r^\dagger a_r+\frac{1}{2}
    \right).
}
定义所有模式共同的真空态 $\ket0$ 满足 $a_r\ket0=0,\forall r$，
则一般激发态可写成
\eq{
    \ket{n_1,n_2,\cdots}
    =
    \prod_r
    \frac{(a_r^\dagger)^{n_r}}{\sqrt{n_r!}}
    \ket0.
}
它的能量为
\eq{
    E_{\{n_r\}}
    =
    \sum_r
    \hbar\omega_r
    \left(
        n_r+\frac{1}{2}
    \right).
}
这里 $n_r$ 表示第 $r$ 个动量模式被激发了多少次，称 $p_r$ 的\textbf{占有数}为 $n_r$。
这正是 Fock 空间的基本雏形：我们可以不再固定波函数所代表的粒子数，而是记录每个模式的占有数。

我们不再追问每个粒子的态，而是追问每个场模式对应含有多少个量子。把所有粒子数的态空间合在一起，得\textbf{Fock 空间}
\eq{
    \mathcal F_\pm(\Hilbert)
    =
    \Hilbert^{(0)}\oplus\Hilbert_\pm^{(1)}
    \oplus\Hilbert_\pm^{(2)}\oplus\cdots,
}
其中 $\Hilbert^{(0)}$ 由所谓的“真空态 $\ket 0$”张成。后文引入产生、湮灭算符时，它们的作用就是改变某个模式的占有数；而玻色子用对易关系，费米子用反对易关系，正是为了实现这里的交换对称性和量子统计。
按动量模式做 Fourier 展开的思想将在场论中反复出现。经典场的每个模式都可看成一个独立振动自由度，量子化之后，这些模式的激发就对应粒子。

\subsection{复标量场}

复标量场相当于两个实标量场的组合 $\phi=\frac{1}{\sqrt{2}}(\phi_1+\i \phi_2)$。
自由作用量可用 $\phi,\phi^\dagger$ 构造（注意算符的先后顺序）：
\[\Lag=\p_\mu\phi^\dagger\p^\mu\phi-m^2\phi^\dagger\phi = \frac 12\p^\mu\phi_1\p_\mu\phi_1 - \frac 12 m^2\phi_1^2 +\frac 12\p^\mu\phi_2\p_\mu\phi_2 - \frac 12 m^2\phi_2^2.\]
运动方程为 $(\square+m^2)\phi=(\square+m^2)\phi^\dagger=0$。
$\phi,\phi^\dagger$ 线性独立，代表两个相互独立的自由度。
上式不仅 $Z_2$、Lorentz 不变，且在\textbf{U(1) 整体相位变换} $\phi' =\e^{\i \theta} \phi$（从而 $(\phi^\dagger)'= \e^{-\i \theta} (\phi^\dagger)$）下不变。称为\textbf{U(1) 整体对称性}。

根据上节内容，通解一定形如
\[
    \tcbhighmath{\phi(x) = \int\frac{\d[3]p}{(2\pi)^3}\frac{1}{\sqrt{2E_{\bm p}}} (a_{\bm p} \e^{-\i px} + b^\dagger_{\bm p} \e^{\i px} ),}\quad px = \sqrt{\bm p^2+m^2}\, t - \bm p\cdot \bm x.
\]
$\phi(\bm{x},t)$ 对应的共轭动量密度是
\begin{equation}
  \Pi(\bm{x},t)
  = \pdv{\mathcal{L}}{(\partial_0\phi)}
  = \partial_0\phi^\dagger
  = \int\frac{\d^3p}{(2\pi)^3}\frac{-\i p^0}{\sqrt{2E_{\bm{p}}}}
     \left(b_{\bm{p}}\e^{-\i p\cdot x}
     -a^\dagger_{\bm{p}}\e^{\i p\cdot x}\right),
\end{equation}
$\phi^\dagger(\bm{x},t)$ 对应的共轭动量密度是
\begin{equation}
  \Pi^\dagger(\bm{x},t)
  = \pdv{\mathcal{L}}{(\partial_0\phi^\dagger)}
  = \partial_0\phi
  = \int\frac{\d^3p}{(2\pi)^3}\frac{-\i p^0}{\sqrt{2E_{\bm{p}}}}
     \left(a_{\bm{p}}\e^{-\i p\cdot x}
     -b^\dagger_{\bm{p}}\e^{\i p\cdot x}\right).
\end{equation}
$\Pi(\bm{x},t)$ 与 $\Pi^\dagger(\bm{x},t)$ 互为厄米共轭。等时对易关系为
\begin{align}
  &[\phi(\bm{x},t),\Pi(\bm{y},t)]=\i\delta(\bm{x}-\bm{y}),
  \qquad
  [\phi(\bm{x},t),\phi(\bm{y},t)]
  =[\Pi(\bm{x},t),\Pi(\bm{y},t)]=0,\notag\\
  &[\phi^\dagger(\bm{x},t),\Pi^\dagger(\bm{y},t)]
  =\i\delta(\bm{x}-\bm{y}),
  \qquad
  [\phi^\dagger(\bm{x},t),\phi^\dagger(\bm{y},t)]
  =[\Pi^\dagger(\bm{x},t),\Pi^\dagger(\bm{y},t)]=0,\notag\\
  &[\phi(\bm{x},t),\Pi^\dagger(\bm{y},t)]
  =[\phi^\dagger(\bm{x},t),\Pi(\bm{y},t)]
  =[\phi(\bm{x},t),\phi^\dagger(\bm{y},t)]
  =[\Pi(\bm{x},t),\Pi^\dagger(\bm{y},t)]=0.
\end{align}
利用这些关系可推出以下产生湮灭算符的对易关系：
\begin{align}
  &[a_{\bm{p}},a^\dagger_{\bm{q}}]=(2\pi)^3\delta(\bm{p}-\bm{q}),
  \qquad
  [a_{\bm{p}},a_{\bm{q}}]=[a^\dagger_{\bm{p}},a^\dagger_{\bm{q}}]=0,\notag\\
  &[b_{\bm{p}},b^\dagger_{\bm{q}}]=(2\pi)^3\delta(\bm{p}-\bm{q}),
  \qquad
  [b_{\bm{p}},b_{\bm{q}}]=[b^\dagger_{\bm{p}},b^\dagger_{\bm{q}}]=0,\notag\\
  &[a_{\bm{p}},b^\dagger_{\bm{q}}]
  =[b_{\bm{p}},a^\dagger_{\bm{q}}]
  =[a_{\bm{p}},b_{\bm{q}}]
  =[a^\dagger_{\bm{p}},b^\dagger_{\bm{q}}]=0.
\end{align}
这些对易关系说明 $(a_{\bm{p}},a^\dagger_{\bm{p}})$ 和 $(b_{\bm{p}},b^\dagger_{\bm{p}})$ 是两套相互独立的产生湮灭算符，描述两种不同的标量玻色子。

对复标量场 $\phi(x)$ 作 $U(1)$ 整体变换
\begin{equation}
  \phi'(x)=\e^{\i q\theta}\phi(x),
  \qquad
  [\phi^\dagger(x)]'=\e^{-\i q\theta}\phi^\dagger(x),
\end{equation}
其中实常数 $q$ 是 $\phi(x)$ 的 $U(1)$ 荷，实数 $\theta$ 是不依赖于 $x^\mu$ 的变换参数，则拉氏量 (2.179) 不变，系统具有 $U(1)$ 整体对称性。相应的 $U(1)$ 守恒流算符为
\begin{equation}
  J^\mu=\i q\phi^\dagger\overleftrightarrow{\partial^\mu}\phi,
\end{equation}
它满足守恒流方程 $\partial_\mu J^\mu=0$。$J^\mu$ 是一个厄米算符，满足
\begin{equation}
  (J^\mu)^\dagger
  = \{\i q[\phi^\dagger\partial^\mu\phi-(\partial^\mu\phi^\dagger)\phi]\}^\dagger
  = -\i q[(\partial^\mu\phi^\dagger)\phi-\phi^\dagger\partial^\mu\phi]
  = \i q\phi^\dagger\overleftrightarrow{\partial^\mu}\phi
  = J^\mu.
\end{equation}
相应的 $U(1)$ 守恒荷算符是
\begin{align}
  Q
  &= \i q\int\d^3x\,\phi^\dagger\overleftrightarrow{\partial^0}\phi
   = \i q\int\d^3x\,[\phi^\dagger\partial^0\phi-(\partial^0\phi^\dagger)\phi]
   = \i q\int\d^3x\,(\phi^\dagger\pi^\dagger-\pi\phi) \notag\\
  &= q\int\frac{\d^3p}{(2\pi)^3}
     (a^\dagger_{\bm{p}}a_{\bm{p}}-b_{\bm{p}}b^\dagger_{\bm{p}}).
\end{align}
% TODO-PDF-PAGE-76: verify the omitted multi-line intermediate expansion in equation (2.211) against the rendered source page.
再利用对易关系 (2.193)，得到
\begin{equation}
  Q
  = \int\frac{\d^3p}{(2\pi)^3}
    q(a^\dagger_{\bm{p}}a_{\bm{p}}-b^\dagger_{\bm{p}}b_{\bm{p}})
    -(2\pi)^3\delta^{(3)}(0)\int\frac{\d^3p}{(2\pi)^3}q.
\end{equation}
上式第二项是零点荷。在第一项的圆括号中，粒子数密度算符 $a^\dagger_{\bm{p}}a_{\bm{p}}$ 的系数是 $q$，而粒子数密度算符 $b^\dagger_{\bm{p}}b_{\bm{p}}$ 的系数是 $-q$。可见，$(a_{\bm{p}},a^\dagger_{\bm{p}})$ 描述的粒子具有的 $U(1)$ 荷为 $q$，称为正粒子；$(b_{\bm{p}},b^\dagger_{\bm{p}})$ 描述的粒子具有相反的 $U(1)$ 荷 $-q$，称为反粒子。因此，复标量场描述一对正反标量玻色子。除去零点荷，总荷 $Q$ 是所有动量模式所有正反粒子贡献的 $U(1)$ 荷之和。这里单个粒子的 $U(1)$ 荷 $q$ 或 $-q$ 对总荷 $Q$ 的贡献是相加性的，并且来自于一种内部对称性，因而是一种内部相加性量子数 (internal additive quantum number)。实际上，

\begin{center}
  任何反粒子的所有内部相加性量子数都与相应的正粒子相反。
\end{center}

% --- PDF page 77, printed page 69 ---
(2.212) 式不依赖于时间 $t$，这是 $U(1)$ 荷守恒定律的体现。

如果将复标量场 $\phi(x)$ 替换成量子力学的单粒子波函数 $\Psi(x)$，则
\begin{equation}
  \frac{Q}{q}
  = \i\int\d^3x\,[\phi^\dagger\partial^0\phi-(\partial^0\phi^\dagger)\phi]
\end{equation}
与单粒子概率密度 (1.3) 的全空间积分类似。但是，这里 $Q/q$ 的本征值被解释为正粒子数与反粒子数之差，显然是可正可负的。也就是说，在量子场论中 $Q/q$ 与单粒子在空间中的概率没有关系，不像量子力学那样存在负概率困难。

如果对实标量场作类似的 $U(1)$ 整体变换，则自共轭条件 (2.78) 使得
\begin{equation}
  \e^{\i q\theta}\phi(x)
  = \phi'(x)
  = [\phi'(x)]^\dagger
  = [\e^{\i q\theta}\phi(x)]^\dagger
  = \e^{-\i q\theta}\phi^\dagger(x)
  = \e^{-\i q\theta}\phi(x).
\end{equation}
上式要求 $q=0$。因此，对实标量场不能进行非平庸的 $U(1)$ 整体变换。实际上，自共轭条件使实标量场描述的粒子不能具有任何非零的内部相加性量子数，也就是说，正粒子与反粒子是相同的。这样的粒子称为纯中性粒子 (truly neutral particle)，而实标量场是一种纯中性的场。

经过进一步计算，得到自由复标量场的哈密顿量算符
\begin{equation}
  H
  = \int\frac{\d^3p}{(2\pi)^3}E_{\bm{p}}
    (a^\dagger_{\bm{p}}a_{\bm{p}}+b^\dagger_{\bm{p}}b_{\bm{p}})
    +(2\pi)^3\delta^{(3)}(0)\int\frac{\d^3p}{(2\pi)^3}E_{\bm{p}}.
\end{equation}
除了零点能，哈密顿量是所有动量模式所有正反粒子贡献的能量之和。动量为 $\bm{p}$ 的正粒子和反粒子的能量都是 $E_{\bm{p}}=\sqrt{|\bm{p}|^2+m^2}$，可见，正反粒子具有相同的质量 $m$。另一方面，复标量场的总动量算符为
\begin{equation}
  \bm{P}
  = \int\frac{\d^3p}{(2\pi)^3}
    \bm{p}(a^\dagger_{\bm{p}}a_{\bm{p}}+b^\dagger_{\bm{p}}b_{\bm{p}}).
\end{equation}
总动量是所有动量模式所有正反粒子贡献的动量之和。具体推导过程见 2.4.4 小节选读内容。

    \Delta_F(x-y)
    :=
    \ev{T\{\phi(x)\phi(y)\}}{0}
}
称为\textbf{Feynman 传播子}。它与前面由 $\hat L^{-1}$ 得到的 Green 函数是同一类对象：除去常数约定外，都是某个线性算符的逆，只是还必须指定边界条件。路径积分的真空时序二点函数自动选出 Feynman 边界条件，这也是它在QFT中尤其方便的原因。
