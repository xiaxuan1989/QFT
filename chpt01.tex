\section{量子力学回顾}

近代光学实验中，光常同时表现出粒子和波动的性质，人们逐渐形成波粒二象性的观念，物质状态笼统地称为\textbf{量子态}(quantum state)。
量子一词起初源于黑体辐射的能量离散化，后来延伸为波粒二象性等有悖于经典状态的性质。
经典理论中单粒子系统由位置及动量 $(\bm x,\bm p)$ 描述，系统的运动视作相空间中的轨迹；但有趣的是，在量子理论中，我们无法同时观测粒子位置及动量。
Heisenberg 提出，经典的测量值 $\bm x,\bm p$ 是实数，但二者的不确定性意味着必须将其“提升”为一种\textbf{算符}(operator) $\hat{\bm x},\hat{\bm p}$，作用于物质的量子态上。
考虑一维情形，他根据实验和已有研究提炼出
\eq{
    [\hat x,\hat p]:=\hat x\hat p - \hat p\hat x = \i\hbar,
}
$\hbar=h/2\pi$ 是约化 Planck 常数。这称为 $\hat x,\hat p$ 的\textbf{对易子}，意思无非是对量子态先作用 $\hat p$ 再作用 $\hat x$，减去其相反操作，结果相当于乘上 $\i\hbar$。所以量子力学不可避免涉及复数。
尽管我们仍没有说出量子态、算符的数学表示，但我们要求它们一定服从这种对易关系。
经典力学是 $\hbar\to 0$ 的极限，意味着 $\hat x\hat p=\hat p\hat x$，作为普通实数的 $x,p$ 就满足这个可交换乘法。
算符的非对易性可以说是量子性质的关键，因此这种“算符提升”也称为从经典力学走向量子力学的\textbf{量子化}。

下面我们来猜测算符可能的数学表示。人们现已广泛清楚，矩阵的乘法是不对易的，因此矩阵是合适的数学候选。如此，量子态就是列矢量，写作 $\ket{}$。一般会在里面放一些符号，仅表标签含义，比如 $\ket{\psi}$。全体 $\ket{\psi}$ 构成线性空间。
但在当时，矩阵并非必学知识。Heisenberg 从经典力学的 Poisson 括号绕了一圈才回到了矩阵，相当于重新发明了矩阵。可以说，今日之大学得以普及线性代数，有 Heisenberg 的功劳。
不过这里会比我们学的线性代数复杂：因为可以证明，任意有限维度的矩阵都无法凑出对易关系。因此，算符必须是无限维空间上的矩阵。
相较于 Heisenberg，Schr\"odinger 更关注于量子态的具体表示，方法会更简单些。
他考虑的算符是直接的偏微分算符，比如 $\grad,\pdv{t}$。因此量子态可以表示为某种实变但复值的函数 $\psi(\bm x,t)$，称为\textbf{波函数}。
后面会证明两种观点等价，或者说 $\ket{\psi},\psi(\bm x,t)$ 有对应关系。
这也解释了为什么 $\ket{\psi}$ 线性空间是无穷维的，因为 $\psi(\bm x,t)$ 需要遍历不可数的 $\bm x,t$，即函数空间是无穷维的。

由于现在提升为算符，可能的测量值只能解释为算符的某个\textbf{实数}本征值。如果 $\hat A$ 具有分立的本征值 $\{a_n\}$，称 $\hat A$ 具有\textbf{离散谱}，测量 $A$ 的结果只能是 $\{a_n\}$ 中的一个。回忆一下，本征值的定义无非是 $\hat A \ket{a_n}=a_n \ket{a_n}$，对应的 $\ket{a_n}$ 称为 $\hat A$ 的本征矢。
Born 根据杨⽒双缝实验提出，测量结果 $a_n\in\R$ 出现的概率由态决定：若 $\ket\psi=\sum_n c_n \ket{a_n}$，则相应概率为 $|c_n|^2$。
这可解释为 $\ket{\psi}$ 在 $\ket{a_n}$ 的投影或者说内积结果。
由于我们现在考虑的是复线性空间，内积结果为复数，所以定义方式会区别于实线性代数，满足：
\begin{itemize}
    \item $\braket{a}{b}=\bra{b}\ket{a}^*$（复共轭）。
    \item $\braket{a}{a}\geq 0$，且 $\braket{a}{a}=0\iff \ket{a}=0$。
    \item $\braket{a}{b+c}=\braket{a}{b}+\braket{a}{c}$。    \item $\braket{a}{\lambda b}=\lambda \braket{a}{b}$。
\end{itemize}
后两个条件表明对第二槽线性，但结合第一条可知对第一槽\textbf{反线性}：$\braket{\lambda a}{b}=\lambda^* \braket{a}{b}$。
再补充一些条件后，这个空间成为了俗称的\textbf{Hilbert 空间}，是量子态的自然载体。
由此，如果要求可观测量 $\hat A$ 是自伴的
\eq{
    \hat A = \hat A^\dagger,
}
其中 $\hat A^\dagger$ 是 $\hat A$ 的\textbf{共轭转置}。简单计算会发现，自伴算符的本征矢是正交归一基
\eq{
    \bra{a_n}\ket{a_m} = \delta_{nm}.
}
并且对于分立的本征值，内积定义相当于给出
\eq{
    \bra{\psi} := \sum_n c_n^* \bra{a_n} = {\ket{\psi}}^\dagger,\quad \braket{\psi}{\psi} = \sum_n c_n^* c_n = \sum_n |c_n|^2.
}
于是
\eq{
    \bra{a_n}\ket{\psi} = \sum_m c_m \delta_{nm} = c_n.
}
因此测量结果 $a_n$ 出现的概率为 $|\bra{a_n}\ket{\psi}|^2$。总和为 $\sum_n |\bra{a_n}\ket{\psi}|^2=1$，等价为归一化条件 $\braket{\psi}{\psi}=1$。
算符 $\hat A$ 的平均值可以写作
\eq{
    \langle A \rangle := \sum_{n} a_n |\bra{a_n}\ket{\psi}|^2 = \bra{\psi}\hat A\ket{\psi}.
}

推广到连续本征值时，称这个算符具有\textbf{连续谱}，位置算符 $\hat{\bm x}$ 就是典型例子。我们可以关注上述定义或性质中不含 $\sum_n$ 的版本。比如，如果沿用 $\hat{\bm x}\ket{\bm x}=\bm x\ket{\bm x}$，这里 $\ket{\bm x}$ 是 $\hat{\bm x}$ 的本征矢，所以用了标签 $x$，仅此而已。则 $\braket{\bm x}{\psi}$ 表示测量粒子（$t$ 时刻）发现在 $\bm x$ 的概率密度分布，因此
\eq{
    1 = \int\d[3]{x} |\braket{\bm x}{\psi}|^2,\quad \langle \bm x \rangle = \bra{\psi}\hat{\bm x}\ket{\psi} = \int\d[3]{x} \bm x |\braket{\bm x}{\psi}|^2.
}
现在态矢的展开就写为
\eq{
    \ket{\psi} = \int\d[3]{x} \ket{\bm x}\braket{\bm x}{\psi}.
}
可见 Schr\"odinger 表述的等价性就体现在
\eq{
    \psi(\bm x,t) = \braket{\bm x}{\psi}.
}
即波函数是 $\ket{\psi}$ 在 $\ket{\bm x}$ 上的分量。
波函数的内积都写作
\eq{
    \braket{a}{b} = \int\d[3]{x} a^*(\bm x) b(\bm x).
}
算符的平均值为 $\langle A \rangle = \int\d[3]{x} \psi^* \hat A \psi$。

位置平均值为 $\langle \bm x \rangle=\int\d[3]{x} \psi^*(\bm x) \bm x\psi(\bm x)$。
这意味着不仅 $\hat{\bm x}$ 是本征矢，实际上 $\psi$ 的每个位置分量 $\psi(\bm x)$ 都是 $\hat{\bm x}$ 的本征矢，且本征值为 $\bm x$。位置算符的平均值为位置平均值，则必须 $\hat{\bm x} = \bm x$。
由 Leibniz 乘法法则得 $\hat p : = -\i\hbar\grad$ 满足对易关系。

量子态的时间演化由哈氏算符生成，或等价地
\eq{
    \i\hbar\pdv{t}\ket{\psi(t)} = \hat H \ket{\psi(t)},\quad \hat H = \frac{\hat{\bm p}^2}{2m} + V(\hat{\bm x},t).
}
位置表象下展开，说明波函数应满足方程
\eq{
    \i\hbar\pdv{t}\psi(\bm x,t)=\left(-\frac{\hbar^2}{2m}\Delta+V(\bm x,t)\right)\psi(\bm x,t).
}
最初 Schr\"odinger 通过波动力学的一些类比方法得到。



可见若 $\{\psi_i\}$ 是解，则任意\textbf{线性叠加} $\sum_i c_i\psi_i$ 也是解。
这也意味着 $\psi$ 的重标度 $c\psi$ 仍是解，故总可以对波函数归一化。
这与 $\ket{\psi}$ 的线性空间性质相符。
$V$ 为常数时，上式具有简单的平面波解 $\psi=A\e^{\i(\bm k\vdot\bm x - \omega t)}$。容易发现 $\bm k,\omega$ 必须满足 $\omega = \hbar \bm k^2/2m + V/\hbar$。借 Planck 黑体辐射、Einstein 光电效应，de Broglie 指出，粒子的能量、动量可与波函数的波动性质相联系：
\eq{
    E = h\nu = \hbar\omega,\quad \bm p = \hbar\bm k.
}
因此这无非是说粒子满足经典力学而非相对论的关系式
\eq{
    E = {\bm p^2}/{2m} + V.
}

这里平面波之所以重要，是因为它们是动量算符的本征函数。一般波函数未必本身是平面

$[-L/2,L/2]$ 上良好的函数可展开为 Fourier 级数
\[f(x)=\sum_{N=-\infty}^\infty \frac{F(p_N)}{L} \e^{\i p_N x},\quad F(p_N)=\int_{-L/2}^{L/2}\d{x}\e^{-\i p_N x}f(x),\]
其中 $p_N={2\pi N}/{L}$ 称为\textbf{模式}(mode)，$F(p)$ 是 $f(x)$ 的 Fourier 变换。令 $L\to\infty$，则 $\Delta p=2\pi/L\to 0$，级数化为积分，即 $f(x)=\int \frac{\d p}{2\pi} \e^{\i p x}F(p),F(p)=\int\d{x}\e^{-\i p x}f(x)$。任意 $f(x)$ 以 $\e^{\i px}$ 为基底。
同理，取 $V=L^n$ 并连续化，可得
\[f(x)=\int \frac{\d[n]p}{(2\pi)^n} \e^{\i p_\mu x^\mu}F(p),\quad F(p)=\int\d[n]{x}\e^{-\i p_\mu x^\mu}f(x).\]
这种按动量模式展开的思想将在场论中反复出现。经典场的每个Fourier 模式都可看成一个
独立振动自由度；量子化之后，这些模式的激发就对应粒子。

当有多个粒子时，比如两个粒子，系统的量子态 $\psi(\bm x_1,\bm x_2,t)$ 是两个位置变量的函数。对于 $N$ 个粒子，系统的量子态 $\psi(\bm x_1,\cdots,\bm x_N,t)$ 是 $3N$ 个位置变量的函数。因此，粒子数的增加导致系统的量子态空间维数呈指数级增长。

波函数不再是单个粒子的位置的函数，而是所有粒子位置的函数。这种描述方式虽然在理论上是可行的，但在实际计算中非常复杂，尤其是当粒子数目较多时。
引入态矢量的概念后，系统的量子态可以表示为 $\ket{\psi}$，而不是具体的波函数形式。这种抽象的表示方式使得我们能够更灵活地处理多粒子系统，尤其是在涉及到粒子交换对称性和量子统计时。

因此普通量子力学通常先固定粒子数，再讨论这个固定粒子数Hilbert 空间中的态。然而在
相对论物理中，粒子可以产生和湮灭，固定粒子数的描述不再自然。量子场论的基本想法
是：不把粒子的位置波函数作为最基本对象，而是把场本身量子化；粒子则被解释为量子
场的激发。

因此，描述多粒子系统的量子态空间非常庞大，难以直接处理。为了解决这个问题，物理学家引入了\textbf{量子场}的概念，将粒子视为量子场的激发态，从而能够更有效地描述多粒子系统的行为。这种方法不仅简化了多粒子系统的描述，还为理解粒子之间的相互作用提供了一个更自然的框架。

因此，量子力学给出了后文所需的两条主线：一是将经典物理量提升为非对易算符，二
